# MoneyLion Transaction Categorization

This project predicts bank transaction categories using machine learning, based on transaction descriptions and user profile data.

## Contents
- `moneylion_model_notebook.ipynb` — Jupyter Notebook with model training and evaluation
- `moneylion_model_notebook.html` — Exported HTML version
- `vizualisations/*.png` — Visualizations of model performance

## Notes
- Original datasets (`bank_transaction.csv`, `user_profile.csv`) are **not included** per challenge instructions.
